# Image Setup

To complete the landing page, add these images to the `public` folder:

1. `tradition.jpg` - An image showing traditional wellness (like herbs, natural ingredients)
2. `science.jpg` - An image showing modern science (like lab equipment, molecules)

You can use placeholder images from:
- https://unsplash.com
- https://pexels.com
- Or any stock photo service

The images should be:
- High quality (at least 1200x800px)
- Relevant to wellness/health themes
- Professional looking

Once added, the hero section will display them automatically.
